# Loops in C#

## What is a Loop?
A loop is used to repeat a block of code multiple times until a condition is met.

## Types of Loops
- for loop
- while loop
- foreach loop

## Example
```csharp
using System;

class Program {
    static void Main() {
        for (int i = 1; i <= 5; i++) {
            Console.WriteLine(i);
        }
    }
}
